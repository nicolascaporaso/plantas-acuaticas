Ficha Técnica: Helecho de Java

Descripción

El helecho de Java es una planta de acuario popular y muy extendida. Es ideal para principiantes porque es robusta, fácil de cuidar y capaz de adaptarse. Una particularidad de esta planta es que las raíces se pegan fácilmente a los objetos, como las piedras. A continuación, te explicamos todo lo que debes saber sobre el Microsorum pteropus, desde el cuidado y la tenencia hasta la plantación correcta.

El helecho de Java es una planta helófita que procede de Asia. En la naturaleza, el Microsorum pteropus crece en arroyos y cascadas, principalmente sobre rocas o pegada a la madera. Esta planta tiene una enorme capacidad de adaptación y puede crecer tanto bajo el agua como en tierra sobre suelo boscoso húmedo. Las hojas son alargadas y se estrechan hasta terminar en punta, mientras que la raigambre varía de color entre marrón oscuro y negro.

Ficha

  • Nombre científico: Microsorum pteropus
  • Familia: Polipodiáceas (Polypodiaceae)
  • Origen: Trópico y subtrópico asiáticos
  • Dificultad: Apta para principiantes, fácil de cuidar y robusta
  • Hojas: Más delgadas que anchas, de un verde intenso y duras
  • Altura máxima: Entre 15 y 30 cm (dependiendo de la especie)
  • Crecimiento: Lento
  • Raíces: De marrón oscuro a negro
  • Temperatura del agua: De 20 a 28 °C
  • Luz: Puntos de sombra y luz posibles
  • Ubicación en el acuario: Las especies bajas se pueden poner en la parte delantera del acuario, mientras que las grandes deben plantarse detrás.
  • Particularidades: Como planta epífita, el helecho de Java crece mejor extendido sobre piedras o raíces en el sustrato.
  • Apta para: Recipientes con percas, acuarios de gambas y como planta de terrario en un paludario (acuaterrario).
  • Importante: Los rizomas (tallos subterráneos) no deben ser enterrados, ya que se pudren.

¿Qué especies hay?

Existen varias variantes de helechos de Java que puedes adquirir para tu acuario. Aquí te presentamos las formas de cría más populares:

  • Helecho de Java Tropica: (Microsorum pteropus «Tropica»): Puede alcanzar hasta 30 cm, a veces incluso 50 cm. Ideal para acuarios grandes.
  • Helecho de Java Windelov: (Microsorum pteropus «Windelov»): Variante más pequeña, que alcanza entre 10 y 20 cm de altura.
  • Helecho de Java Narrow Leaf: (Microsorum pteropus «narrow leaf»): Con hojas más estrechas; existen subespecies como el «orange narrow leaf» y el «short narrow leaf», popular en nanoacuarios.
  • Helecho de Java Needle Leaf: (Microsorum pteropus «needle leaf»): Con hojas aún más estrechas que la «narrow leaf».
  • Helecho de Java Trident: (Microsorum pteropus «trident»): Se asemeja a un tridente en forma.

¿Cómo se planta el helecho de Java?

Puedes plantar el helecho de Java en la parte trasera o intermedia del acuario. Aquí hay dos métodos:

  • Plantación en el fondo: No se debe enterrar el rizoma. Fija las raíces en la grava gruesa sin cubrir el rizoma.
  • Atadura: Ata la planta a un objeto (raíces, piedras) con sedal o hilo. Esto permite un diseño atractivo en el acuario.

¿Qué debo tener en cuenta para el cuidado y la tenencia del helecho de Java?

El helecho de Java es robusto y tiene buena capacidad de adaptación. Las dificultades de tenencia son poco frecuentes.

  • Temperatura: Entre 20 y 28 °C.
  • Iluminación: Se adapta a zonas de luz o sombra.
  • Requisitos de agua: No tiene requisitos especiales en cuanto a dureza, pH ni nutrientes, pero un buen aporte de nutrientes y CO₂ mejora su color verde intenso.

Nota: Si aparecen manchas negras en las hojas, puede ser por déficit de hierro o potasio. Utiliza un abono especial y retira las hojas viejas.

¿Cómo se reproduce el helecho de Java?

La reproducción es fácil. Puedes arrancar una parte del rizoma o esperar a que la planta madre forme plantas hijas. Las plantas hijas se reconocen por los esquejes que se forman en las puntas de las hojas.

Conclusión

El helecho de Java, o Microsorum Pteropus, es una hermosa planta acuática de bajos requerimientos. Su adaptabilidad y belleza lo convierten en una elección perfecta para principiantes en el mundo del acuarismo. Además, contribuirá a crear un ambiente equilibrado y armonioso en tu acuario.
